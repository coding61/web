# 程序媛公开代码

## 各分支介绍
```
master   正式服所在分支，也就是用户所看到的页面（所以要保证没问题了，再将自己分支的东西合并过来）
develop  测试服所在分支，便于开发人员在这里测试自己的功能 (将自己分支的东西合并过来测试)
chen_develop 开发人员(Eleven)本地分支，在开始一个新功能是，先在自己分支上写，没问题再合并到其他分支(如测试分支，主分支) (自己分支内容要与主分支 master 保持一致)
club     俱乐部，老早之前写的，写了一半，中间接收到领导指示，停工，目前废弃了
develop_branch 以前的测试服所在分支，不知什么原因(好像某个开发人员写的某个改版网站，停用了)，它已被废弃，现在是 develop
```
